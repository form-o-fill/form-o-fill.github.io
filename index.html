Form-O-Fill

The programmable form filler for developers

Form-O-Fill is a programmable form filler for the Chrome Browser.

And there is more:

  • Squeal with glee as Form-O-Fill report form filling errors back to you
  • Use a function to fetch data dynamically at the start of the rule executing process
  • Use static values or functions as the value for the field
  • Value functions can use the data fetched before!
  • Colorful and friendly help integrated
