Form-O-Fill was removed from the chrome store.
The chrome store "team" has decided to remove Form-O-Fill from the store because it required too many and too broad permissions.
On 7th of march the extension was removed even though the permission were changed beforehand as required. Since it is impossible to get helpful information from the store "team" I decided to not publish this extension in the store anymore.
I'm sorry for those still using it but coping with the store system of allowing or disallowing extensions feels too tedious for now. The extension still works as expected if you build and install it locally.

Follow these instructions to install it locally if you want.

Form-O-Fill is a programmable form filler for the Chrome Browser.

And there is more:

  • Integrated error reporting
  • Integrated live tutorials
  • Use a function to fetch data dynamically at the start of the rule executing process
  • Use static values or functions as the value for the field
  • Value functions can use the data fetched in before functions!
  • Complete help system integrated
  • Automate multiple pages in sequence with workflows
  • Export your rules as JSON so other colleagues can use them
  • Advanced features like custom library functions or shared rules
