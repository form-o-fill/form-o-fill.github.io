From devs for devs

Form-O-Fill is a chrome extension for filling out forms on a website.
It was born out of the need to quickly and repeatedly fill out the same form over and over again during frontend development.
Since then it has learned a lot of cool tricks to make the development of form-heavy sites more easy.
Oh, and it also works with non-forms :)

Driven by JavaScript

In Form-O-Fill everything is described with JavaScript//ECMAScript.
A rule describes on which page to use which field definitions.
Field definitions describe every field to fill. Simple as that.
This is all done using a simple JSON-like structure.

Data extraction made easy

When you start using Form-O-Fill you will need to extract some basic field info from the page you are using it on.
Extract form data using the exttraction overlay or right click on any form field.
After that follow the notification info to make your extracted data editable in the options panel.
